Dev-Build is a project that is developed so as to explore various tech stacks, features each of them offer and how can we use them to solve a problem. An Engineer is always a problem solver :D, so the project is currently about vizualizing the working of an algorithm by hitting backend stack from front end. The challenge was that we have to monitor the changes done by algorithm to the data close to realtime. The challenge is not much hard though and so the project now houses 3 different sorting algorithms and the binary search algorithm.

These algorithms are visualized with the help of Java backend and Chart Js in front end.

This whole project is built on Spring boot and Angular stack.
