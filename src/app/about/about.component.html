Dev - Build.

Dev-build is a hobby project developed on free time. Currently dev-build helps in understanding sorting algorithms and binary search algorithm. The project uses Chart Js for visualizing the working of an algorithm. Projects Front-End tech stack consists of

  • Angular
  • Chart Js
  • Bootstap
  • Highlight Js

Applications backend is built with spring boot. The backend application exposes various endpoints to sort the data as well as to monitor the current state of the sorted data (partially sorted data).
