Minimum Radius (Binary Search) - Hackerearth


The idea is about calculating the cartesian distance of every provided cordinate from (0,0). Sort the Els based on the cartesian distance. Once sorted calculate the total weight each circle will enclose. Then do the binary search.. you can see the code below. While calculating the cartesian distance beware of overflow use 64bit or 128bit representation for the calculation.
